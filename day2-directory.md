# 目录操作 #
### 查看目录
```
ls                  # 查看文件夹
ls -l               # 查看详细信息（权限、大小、时间）
ls -a               # 显示所有文件（包括隐藏文件）
pwd                 # 显示当前所在路径
```


### 切换目录
```
cd /                # 进入根目录
cd \~                # 回到当前用户家目录
cd 目录名           # 进入指定目录
cd ..               # 回到上一级目录
```


### 目录创建/删除
```
mkdir 目录名        # 创建空目录
mkdir -p a/b/c      # 递归创建多级目录
rmdir 目录名        # 删除空目录
rm -r 目录名        # 递归删除目录及内容
```
