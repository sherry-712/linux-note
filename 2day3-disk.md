# 磁盘管理

## 核心概念
### 1. 基础定义
```bash
# 磁盘：物理硬盘（如 /dev/sda）
# 分区：硬盘的逻辑划分（如 /dev/sda1、/dev/sda2）
# 文件系统：分区格式化后的存储格式（如 ext4）
# 挂载：将分区关联到某个文件夹，才能访问其中数据
# 虚拟内存（swap）：硬盘划出的空间，内存不足时缓存数据（free -h 查看）
# /etc 目录：系统所有配置文件的核心目录
```

### 2. 磁盘查看命令
```bash
# 查看磁盘整体使用情况（人类可读格式）
df -h

# 查看当前目录总大小
du -sh

# 查看当前目录下各文件/文件夹大小
du -sh *

# 查看家目录总大小
du -sh ~

# 查看根目录下一级文件夹大小
sudo du -sh /*

# 查看磁盘分区详细信息
sudo fdisk -l

# 树形结构查看磁盘挂载（更直观）
lsblk

# 查看内存和 swap 使用情况
free -h

# 查看当前挂载状态
mount

# 查看开机自动挂载配置
cat /etc/fstab
```
