### 查看网络接口
```
ip addr              # 查看IP地址、网卡信息（等价ifconfig）
```


### 测试网络连通性
```
ping 目标IP/域名      # 测试连通性（用ICMP协议）
ping 8.8.8.8 -c 3    # 发送3次包后停止
```


### 命令行浏览器/下载
```
curl 网址            # 发送HTTP请求，返回网页源码（不渲染）
curl -O 网址         # 下载文件并保留原文件名

wget 网址            # 下载文件（默认保存为原文件名）
wget 网址/pic.jpg    # 下载指定文件
wget -O 新文件名 网址 # 下载并指定保存文件名
```


### 查看端口与进程
```
ss -tulnp            # 查看TCP/UDP监听端口及对应进程
参数：-t(TCP) -u(UDP) -l(监听) -n(不解析服务名) -p(显示进程，需与-l/-a配合)
```
