# vim编辑 #
### 打开/退出：
```
vim 文件名          # 打开文件（不存在则新建）
```


### 命令模式下：
```
:w                  # 保存
:q                  # 退出
:wq                 # 保存并退出
:x                  # 保存并退出（等价:wq）
:q!                 # 强制退出（不保存）
```


### 模式切换
```
i                   # 进入插入模式（光标前插入）
a                   # 进入插入模式（光标后插入）
o                   # 进入插入模式（下一行插入）
ESC                 # 回到命令模式
v                   # 字符可视化模式
V                   # 行可视化模式
Ctrl+v              # 块可视化模式
```


### 光标移动
```
h                   # 左
j                   # 下
k                   # 上
l                   # 右
0                   # 跳到行首
$                   # 跳到行尾
gg                  # 跳到第一行
G                   # 跳到最后一行
:n                  # 跳到第n行（如:10）
```


### 删除/撤销
```
x                   # 删除光标所在字符
dd                  # 删除整行
dw                  # 删除一个单词
u                   # 撤销上一步操作
```


### 复制粘贴
```
yy                  # 复制当前整行
yw                  # 复制一个单词
p                   # 粘贴到下一行
P                   # 粘贴到上一行
v + 移动光标 + y    # 复制选中区域
```


### 查找替换
```
/关键词              # 向下查找关键词（n下一个，N上一个）
?关键词              # 向上查找关键词
:%s/旧/新/g          # 全文替换
:s/旧/新/g           # 替换当前行的所有匹配
```
